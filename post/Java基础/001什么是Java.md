## 维基百科中的Java

> [维基百科中的Java](https://en.wikipedia.org/wiki/Java_(programming_language))

Java是一个高层次的(high-leve)、以类为基础的(class-based)、面向对象的(object-oriented)编程语言。
Java的宗旨是一次编写，随处运行。也就是编译后的Java代码可以在所有支持Java的平台上运行，无需再重新编译。
